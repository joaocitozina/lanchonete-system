# Lanchonete System

## 📝 O Problema
Minha mãe gerencia as vendas de lanches de forma manual, o que dificulta o controle de caixa e estoque. Este sistema automatiza esses processos, trazendo organização financeira e agilidade no atendimento.

## 🛠 Tecnologias
- C#
- .NET 8
- SQLite
